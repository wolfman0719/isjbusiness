# ISJBUSINESS

ISJBUSINESSのDocker Build

## ビルドプロセス

### Build
* `./build.sh`          for Linux, MacOS
* `build.sh`            for Windows

### Run

* `./run.sh`            for Linux, MacOS
* `run.sh`              for Windows

-bash: ./build.sh: Permission deniedとエラーになる場合は、

`chmod 777 *.sh`

を実行後、再度実行
